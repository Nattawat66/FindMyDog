วิธีการทดสอบ & เทคนิค

  • ระบบใช้ KNN โดยตั้งค่า n_neighbors = 3 (ดูเพื่อนบ้านที่ใกล้ที่สุด 3 จุด)
  • ข้อมูลที่ใช้คือเวกเตอร์ embedding_binary ของรูปสุนัขแต่ละตัวในฐานข้อมูล
  • กราฟ t-SNE ใช้สำหรับลดมิติและดูการกระจายตัวของกลุ่มสุนัขใน 2 มิติ
  • Confusion Matrix แสดงการทำนายถูก/ผิดของ KNN โดยแกน Y คือ Dog ID จริง และแกน X คือ Dog ID ที่โมเดลทำนาย
  • ค่าบนเส้นทแยงมุม (Diagonal) ยิ่งสูง แปลว่าโมเดลจำแนกสุนัขตัวนั้นได้ดี
  • ถ้าอยากทดสอบใหม่ ให้เพิ่ม/อัปเดตรูปสุนัขให้มี embedding แล้วกดปุ่มทดสอบอีกครั้ง
